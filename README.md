# 💜 HEE:NATION

> *"Some memories deserve more than a place in our hearts—they deserve a place in code."*

HEE:NATION is a fan-made tribute website dedicated to **Lee Heeseung**. Built with **HTML, CSS, and JavaScript**, this project combines front-end development with heartfelt storytelling, celebrating unforgettable memories, music, and performances that continue to inspire me as an ENGENE.

## ✨ Features

- 🏠 Beautiful Hero Banner
- 💜 Emotional Welcome Section
- 🎂 Live Birthday Countdown (15 October)
- 🖼️ "The Memories I Treasure Most" Photocard Gallery
- 🎵 Favorite Songs Collection
- 🎥 Favorite Fancams
- 🌙 Elegant Footer

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
